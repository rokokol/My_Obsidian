## Базовые

### int()

Переводит строку в целочисленный формат, можно задать заранее базу
```python
print(int(12.1))
print(int('12A', 12)) #Второй параметр - база числа
```

```
12
178
```

### float()

Переводит строку в вещественный вид
```python
print(float('12.1') + 0.3)
```

```
12.4
```

### str()

Преобразует в строку 
```python
print(str(1) + str(2))
```

```
12
```
